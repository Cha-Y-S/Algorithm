### 문제 접근

  - 삼각형의 사이즈가 3 일때, 직접 삼각형을 그려주고, 3보다 큰 경우에는 3개의 분기로 나누어 가면서 삼각형을 그림

    - `draw_star(n, x, y)`

    - `n`: 그리려는 삼각형의 사이즈

    - `x`: 그리려는 삼각형의 상단 꼭짓점 x 좌표

    - `y`: 그리려는 삼각형의 상단 꼭짓점 y 좌표

    ```cpp
    draw_star(n / 2, x, y);
    draw_star(n / 2, x + n / 2, y - n / 2);
    draw_star(n / 2, x + n / 2, y + n / 2);
    ```
