### 문제

N!에서 뒤에서부터 처음 0이 아닌 숫자가 나올 때까지 0의 개수를 구하는 프로그램을 작성하시오.

---

### 입력

첫째 줄에 N이 주어진다. (0 ≤ N ≤ 500)

---

### 출력

첫째 줄에 구한 0의 개수를 출력한다.

---

### 예제 입력

| 예제 입력1 | 예제 출력1 |
| :--------- | :--------- |
| 10         | 2          |

| 예제 입력2 | 예제 출력2 |
| :--------- | :--------- |
| 3          | 0          |

---

### 문제 접근

  - 주어진 N에 대해, 1부터 N까지 순회

  - 해당 수를 소인수 분해 했을때, `2 x 5`의 조합이 있는지 확인

  - 해당 조합이 있는 개수를 모두 더함
